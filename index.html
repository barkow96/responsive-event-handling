WHAT IS THIS SITE ABOUT?

This site is showing basics of event handling using standard JavaScript Library.

  • EVENT BUBBLING is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up (or you can say is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.
  • EVENT CAPTURING is the other way to do event propagation in the HTML DOM. In event capturing, an event propagates from the outermost element to the target element. It is the opposite of event bubbling, where events propagate outwards from the target to the outer elements.

Additionally, you can adjust the event propagation property. In this way you are able to control whether the first encountered event on the path of the DOM tree in the bubbling or capturing phase, or all of them will be executed one by one.

INSTRUCTION

There are 16 different tiles. Each contains 3 nested ractangles: the most nested green one, middle one which is dark and outer, orange one.

You can click on rectangles to gain points:

  • INNER - 5 points
  • MIDDLE - 2 points
  • OUTER - 1 point

You can also perform some actions on tiles:

  • REMOVE - click and choose specific tile to be removed
  • PROPAGATION - on/off the event propagation
  • DIRECTION - toggle the direction of propagation
  • RESET - reset the overall score
